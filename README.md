# 🏊 Pool-Service - Palm Coast Pool Cleaning Services

Isang tumutugon at modernong website para sa isang negosyo ng paglilinis at pagpapanatili ng mga swimming pool sa Palm Coast, Florida. Ang proyektong ito ay naglalaman ng mga pahina para sa Home, About Us, at Contact Us na binuo gamit ang malinis na HTML at semantic structure.

*(A responsive and modern website for a swimming pool cleaning and maintenance business based in Palm Coast, Florida. This project features Home, About Us, and Contact pages built with clean HTML and semantic structure.)*

---

## 🛠️ Mga Teknolohiyang Ginamit *(Tech Stack)*

* **HTML5** - Para sa semantic na istruktura ng mga pahina.
* **CSS3** - Para sa disenyo, typography, grids, at responsiveness ng layout.
* **JavaScript (ES6)** - Para sa interaktibidad ng navigation menu, back-to-top button, at form validation validation.

---
